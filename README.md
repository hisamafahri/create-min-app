## create-min-app

Generate most minimum projects easily.

### Available Options

- Node.js

  - Base
    - NodeJS + TypeScript
  - Web server
    - Fastify

- Bun
  - Web server
    - Hono

### Usage

```shell
npx create-min-app@latest
```
